The vision

Open a project
containing an .editorconfig file
with your favorite IDE/editor

…​and the TABs/spaces, indentation size, etc…​
will be configured for you automagically

An .editorconfig file

root = true

[*]
charset = utf-8
indent_style = space
indent_size = 4
trim_trailing_whitespace = true
end_of_line = lf

[*.xml]
indent_size = 2

[*.cmd]
end_of_line = crlf
